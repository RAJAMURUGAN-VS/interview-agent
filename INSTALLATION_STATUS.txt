================================================================================
                        TECHNICAL DETAILS
================================================================================

OCR Architecture:
  PDF Upload → PyPDFLoader (text) → pdf2image (images) → EasyOCR (text)
             → Combined + Embedded → Chroma Store → Similarity Search

Fallback Strategy:
  1. Try: pytesseract + Tesseract (if system-installed)
  2. Fallback: easyocr (pure Python, no system deps)
  3. Skip: If both unavailable (PDF still works, images skipped)

Models Used:
  - Embeddings: sentence-transformers/all-mpnet-base-v2
  - OCR: EasyOCR (PyTorch-based, auto-downloads on first use)
  - LLM: Perplexity Sonar (for PDF chat answers)

Performance Notes:
  - First PDF with images: ~30-60s (EasyOCR downloads ~300MB)
  - Subsequent PDFs: ~10-20s (models cached)
  - Text-only PDFs: unchanged performance
  - Image extraction scales with page count (10 pages ~15s)
